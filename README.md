# Классификатор красных испанских вин
Проект для предсказания рейтинга испанских вин на основе их характеристик с помощью  ML.
# О датасете:
Датасет содержит информацию о 7,500 различных испанских красных винах с 11 характеристиками:
1. winery: Название винодельни
2. wine: Название вина
3. year: Год сбора урожая
4. rating: Средний рейтинг пользователей (от 1 до 5)
5. num_reviews: Количество отзывов пользователей
6. country: Страна происхождения
7. region: Регион производства
8. price: Цена
9. type: Сорт вина
10. body: Показатель полнотелости (от 1 до 5)
11. acidity: Показатель кислотности (от 1 до 3)
# Установка 
pip install -r requirements.txt
# Запуск
streamlit run TIPIS.py
